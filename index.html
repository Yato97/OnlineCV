Système de codage/décodage de résultats d’évaluations en ligne

Faculté Sciences Technologies, Université de la Réunion

La conception d'un système anti-fraude pour les évaluations réalisées hors d'une plateforme sécurisée comportant :
- Un module installable comportant un formulaire permettant aux étudiants de récupérer leurs notes sous la forme d'une clé chiffré côté serveur.
- Un logiciel permettant aux professeurs de déchiffrer les clés fournit par les étudiants et de les entrer dans la base de données.
Langages utilisés : HTML, CSS, JS, NODE.JS, JAVA
